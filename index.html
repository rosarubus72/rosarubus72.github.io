Abstract

Voxel-based methods have achieved state-of-the-art performance for 3D object detection in autonomous driving. However, their significant computational and memory costs pose a challenge for their application to resourceconstrained vehicles. One reason for this high resource consumption is the presence of a large number of redundant background points in Lidar point clouds, resulting in spatial redundancy in both 3D voxel and BEV map representations. To address this issue, we propose an adaptive inference framework called Ada3D, which focuses on reducing the spatial redundancy to compress the model's computational and memory cost. Ada3D adaptively filters the redundant input, guided by a lightweight importance predictor and the unique properties of the Lidar point cloud. Additionally, we maintain the BEV features' intrinsic sparsity by introducing the Sparsity Preserving Batch Normalization. With Ada3D, we achieve 40% reduction for 3D voxels and decrease the density of 2D BEV feature maps from 100% to 20% without sacrificing accuracy. Ada3D reduces the model computational and memory cost by 5√ó, and achieves 1.52√ó/1.45√ó end-to-end GPU latency and 1.5√ó/4.5√ó GPU peak memory optimization for the 3D and 2D backbone respectively.

Method

The proposed method, Ada3D, addresses the spatial redundancy in both 3D voxel and 2D BEV features to enhance the efficiency of voxel-based 3D detection methods. Ada3D introduces an adaptive inference framework that selectively filters out redundant 3D voxels and 2D BEV features during inference. Specifically, it employs a lightweight predictor to assess the importance of input features in the BEV space, combining this with the density of the Lidar point cloud to determine which features to drop. This adaptive filtering significantly reduces computational and memory costs by up to 5√ó. Additionally, Ada3D introduces a sparsity-preserving batch normalization technique that maintains sparsity in 2D BEV features, further optimizing model efficiency. By adaptively skipping redundant features, Ada3D achieves a 1.4√ó end-to-end speedup and a 2.2√ó reduction in GPU peak memory usage on an RTX3090, without compromising performance. This approach allows for finer voxel sizes, thereby improving model performance under resource constraints.

Experiments

Experiments and Key Findings: The research conducted several experiments to evaluate the effectiveness of the Ada3D method in enhancing 3D object detection. Notably, Ada3D was tested against various state-of-the-art methods on the KITTI dataset, focusing on metrics such as mean Average Precision (mAP) and 3D detection accuracy across different IoU thresholds.

Performance Comparison: Table 9 and Table 10 demonstrate the performance of Ada3D when combined with the CenterPoint model. Specifically, the method achieved a mAP of 65.5, comparable to the baseline CenterPoint model (66.1) while enabling a finer voxel size (0.075) versus the larger size (0.1) typically used to reduce computational costs. Additionally, Table 10 shows that Ada3D improved performance metrics for cars, pedestrians, and cyclists at various IoU thresholds, indicating enhanced detection accuracy without significant increases in computational cost.

Adaptive Inference and Spatial Redundancy Reduction: Ada3D also integrates adaptive inference techniques to reduce spatial redundancy in both 3D and 2D backbones. By employing a spatial filtering module that combines predictor scores with 3D point cloud density, the method selectively drops redundant parts. Furthermore, the introduction of SP-BN (Spatially Pruned Batch Normalization) helps in retaining sparsity by omitting background pixels in the 2D backbone. These experiments highlight that Ada3D not only maintains or improves detection accuracy but also reduces spatial redundancy, making it a promising approach for efficient 3D object detection.
